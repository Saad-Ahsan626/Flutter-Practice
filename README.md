# 🚀 Flutter Practice Lab

Welcome to my **Flutter Learning Hub**! This repository serves as my personal sandbox for mastering mobile development. It’s a collection of UI challenges, logic implementations, and full-scale features designed to bridge the gap between a **Computer Science student** and a professional **AI Developer**.

As I progress, this lab evolves from simple layouts to complex applications that integrate state management, local/cloud databases, and eventually, **Generative AI**.

---

### 🧪 What’s Inside?
* **State Management Experiments:** Practical implementations of **Provider** and **Cubit** to handle app data flow efficiently.
* **Clean Architecture:** Practicing the separation of UI, Business Logic, and Data layers for scalable code.
* **Backend Integration:** Real-time synchronization and user authentication using **Firebase**.
* **Native Features:** Exploring device-specific capabilities like camera, local storage (SQFlite), and push notifications.
